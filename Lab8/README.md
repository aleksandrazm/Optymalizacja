# Pułkownik Blotto

Program liniowy dla gry Colonel Blotto.
Zakładamy, że każdy z graczy ma 6 oddziałów do rozdysponowania na 3 przełęczach. Gracze mogą wysyłać puste oddziały, jednak w kolejności 
rosnącej, zatem mamy 7 strategii: (0,0,6), (0,1,5), (0,2,4), (0,3,3), (1,1,4), (1,2,3), (2,2,2).

Program można uruchomić na stronie: https://cocalc.com/projects (dawnej: http://cloud.sagemath.org)
